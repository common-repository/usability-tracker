Does your website has usability problems? By installing the Usability Tracker we will check if your visitors encounter usability problems. 

== Description ==

The plugin will load a JavaScript to you page from our servers. This script will track page views and mouse clicks from your visitors. This data is stored in an anonymous way on our servers. 

This data will be analyzed by our servers and the system will create a report of usability problems. This report will be sent to you or can be viewed at our portal at https://www.usabilitytracker.com

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your API key (Which can be requested at https://usabilitytracker.com) in the plugin settings.
